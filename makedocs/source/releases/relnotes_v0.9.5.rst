==================================
lifelib v0.9.5 (18 February 2024)
==================================

In this release, changes have been made to modelx models included in lifelib
to address the backward-incompatible changes introduced in modelx 0.25.0.

.. seealso::

   * `The release note for modelx 0.25.0 <https://docs.modelx.io/en/latest/releases/relnotes_v0_25_0.html>`_ addressing the incompatible changes
   * `The commit reflecting changes <https://github.com/lifelib-dev/lifelib/commit/006e7b0bffa18285076592ce2a573acc510122a8>`_

To update lifelib, run the following command::

    >>> pip install lifelib --upgrade

If you're using Anaconda, use the ``conda`` command instead::

    >>> conda update lifelib
